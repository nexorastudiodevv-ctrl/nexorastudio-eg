# فحص شامل + إصلاح التعارضات (Checklist)

## 1) تشخيص التعارضات (تم مبدئيًا)
- [x] كشف أن الصفحات تربط `script.js.bak` بدل `script.js` (تعارض في المسار)
- [x] كشف أن خادم Express يقدّم الملفات من جذر المشروع (لذلك المسارات حساسة جدًا)

## 2) تعديل المسارات وتوحيدها
- [ ] توحيد جميع الصفحات لاستخدام `/script.js` (إن كان موجودًا فعليًا) أو اعتماد `/script.js.bak` بشكل نهائي
- [ ] توحيد `style.css` بحيث كل الصفحات تستخدم `/style.css` وليس `style.css.bak`

## 3) التحقق الوظيفي
- [ ] التحقق من تحميل CSS و JS بعد النشر (فتح الرابط مباشرة لـ /style.css و /script.js)
- [ ] اختبار `posts.html` وتحميل `news-data.json`
- [ ] اختبار `article` route: /article?id=...

## 4) إصلاح مشاكل منطقية/أخطاء محتملة
- [ ] فحص admin endpoints مقابل admin.html (fetch paths + auth)
- [ ] التأكد من أن صفحة articleDetail تخدم الترتيب الصحيح للمحتوى وتمنع XSS قدر الإمكان

## 5) اختبار نهائي
- [ ] تشغيل `npm start` وفتح أهم الصفحات يدويًا
- [ ] تشغيل Lighthouse/اختبار يدوي سريع
