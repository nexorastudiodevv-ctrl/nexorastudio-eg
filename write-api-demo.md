# ما تم عمله (بدون AI)

- تم إضافة صفحة: `write.html`
- هدفها: إدخال عنوان/محتوى المقال ثم تُخرج لك JSON جاهز.
- لأنك اخترت (3: بدون AI) ولأن السيرفر الحالي لا ينفذ حفظ ملفات تلقائيًا، لا يوجد endpoint يقوم بتعديل `news-data.json` تلقائيًا.

## كيف تضيف المقال لقسم الأخبار الآن

1) افتح `write.html` ثم اكتب المقال.
2) اضغط **تصدير JSON**.
3) انسخ الجزء داخل `items[0]` أو انسخ مصفوفة `items`.
4) افتح `news-data.json` وعدّلها يدويًا بإضافة العنصر الجديد داخل المصفوفة.
5) افتح `posts.html` وستظهر البطاقة مباشرة.

## لاحقًا (لو تريد AI)
نضيف endpoint في `server.js` مثل:
- `POST /api/generate`
ثم يحفظ الناتج في `news-data.json` (إذا السماح بالكتابة)
أو يحفظ في database/JSON file يتم تحديثه عبر السيرفر.
